Connect Your Sheet

Follow these steps to connect the Hansen IGA Google Sheet to this dashboard.

Open the Google Sheet in Google Drive, then click File → Share → Publish to web
Under "Link", select the correct sheet tab, change the format dropdown to Comma-separated values (.csv), then click Publish
Copy the URL it gives you — it will look like:
https://docs.google.com/spreadsheets/d/…/pub?gid=0&single=true&output=csv
Paste that URL below and click Load

Optionally, connect a Google Apps Script to mark deliveries back to the sheet. See instructions at the bottom of this screen.

Apps Script Setup (for delivery confirmation)
Open the Google Sheet → click Extensions → Apps Script
Delete any existing code and paste in the script from the hansen-script.gs file provided
Click Deploy → New deployment → type: Web app → Execute as: Me → Who has access: Anyone → click Deploy
Copy the Web App URL and paste it above
